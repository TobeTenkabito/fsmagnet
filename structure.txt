⚡ 与 qBittorrent 的核心差异对比
维度	qBittorrent	本项目
内存缓存	固定默认值	按本机 RAM 25% 动态分配
连接数	保守上限	按 CPU 核数动态计算
磁盘写入	通用策略	批量合并 + 异步 io_uring
协议加密	默认关闭	强制开启绕过 ISP
Peer 剔除	无主动剔除	定期剔除低分 Peer
上传限制	默认不限	限制上传，带宽全给下载
UI 开销	Qt GUI 占资源	纯异步 CLI，零 UI 开销


用户双击 FSMagnet.exe
        │
        ▼
   main.py 启动
        │
        ├─── 子线程: FastAPI (uvicorn) 监听 127.0.0.1:17878
        │         │
        │         └── 加载 core/session.py（libtorrent 初始化）
        │
        ├─── 等待后端就绪（最多15秒）
        │
        └─── PyWebView 打开窗口
                  │
                  └── 加载 http://127.0.0.1:17878
                            │
                            └── Vue3 前端（来自 ui/dist/）
                                      │
                                      └── SSE 订阅实时速度推送
